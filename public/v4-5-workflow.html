⚖️ L3A vs L3B 路由判斷
判斷維度→ L3A sessions_spawn→ L3B ClawTeam Swarm
子任務數量1–2 個≥3 個並行
程式碼隔離需求不需要需要獨立 git worktree
執行時間<30 分鐘長時間 / 多輪迭代
Agent 溝通需求無 / Telegram 即可即時協調（inbox P2P）
典型場景研究報告、腳本修復、單頁面全棧開發、ML 實驗、審計整改
Token 成本低高（多 agent 並行）
