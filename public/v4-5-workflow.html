§3 L3A 輕量執行（sessions_spawn）

完整串接

① auto_dispatch.sh 每 5 分鐘掃 board_tasks WHERE status='待派發'
  → sessions_spawn(task=dispatch_prompt, model=X, label="task-{id}-auto")
  → UPDATE board_tasks SET status='執行中', updated_at=NOW()

② subagent 執行完成
  → bash ~/clawd/scripts/task_complete.sh {ID} "完成摘要"
     └→ UPDATE board_tasks SET status='已完成', result={摘要}
     └→ notify_telegram.sh 1029808355 "✅ 任務 #{ID} 完成"
     └→ 觸發下一輪 auto_dispatch

③ 失敗偵測（heartbeat_stuck_check.sh 每 10 分鐘）
  → 執行中 >5 分鐘 且 session 不存在 → 判定卡住
  → retry_count < 3：UPDATE status='待派發', retry_count+1
  → retry_count = 3：UPDATE status='已失敗' + 通知 William + 回填 L1
⚠️ 上限 2 個同時 subagent（AGENTS.md §16）。超過排隊等待。
