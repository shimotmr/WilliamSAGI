一、技術背景：什麼是 Markerless 動作捕捉

Markerless Motion Capture（無標記式動作捕捉）利用機器學習 + 多台相機，直接從影像推測人體 3D 骨骼關節位置，不需穿動捕服、不需貼反光 marker。

Vicon Shōgun Markerless 系統規格

  • Vanguard 專用相機：1080p @ 120fps，6.5mm 固定焦距，FOV 78°×49°
  • 紅外線感測：減少可見光干擾
  • AI 追蹤引擎：基於 Vicon 40 年人體動捕數據訓練的專有 ML 模型
  • Shōgun 軟體：即時串流 + 錄製回放一致、FBX 匯出、Genlock 同步
  • 混合模式：可同時捕捉無標記人員 + 有標記道具
  • 定位：影視/遊戲前期快速迭代，非精密測量

三、關鍵質疑與驗證

❓ 質疑 1：精度真的夠嗎？

事實：2023 Nature Scientific Reports 系統性回顧指出，markerless 系統的「時間空間參數」已接近 marker 系統，但關節中心位置和關節角度仍不足以滿足臨床應用。

Vicon 的回應：利用 40 年累積的 ML 模型改善，但本質限制（從體表推測骨骼）沒變。

對機器人的影響：人→機器人需經過 retargeting，這一層的誤差（5-10°）遠大於捕捉精度差異（2-3°），所以高精度捕捉的優勢被吃掉。

❓ 質疑 2：Markerless vs Marker 成本效益

Vicon Markerless 和 Vicon Marker 系統價格相當（都 $100K-$200K），但精度更低。Markerless 唯一優勢是「不用穿動捕服」，這對人形機器人應用幾乎沒意義 — 你不會讓機器人穿動捕服。

❓ 質疑 3：人類動作能直接用在機器人上嗎？

不能。這是最根本的問題：

  • 人類肩關節 3 DOF，機器人可能只有 2 DOF
  • 人類和機器人肢體比例完全不同
  • 機器人有扭矩/速度限制，人類的快速動作機器人做不到
  • 必須經過 Motion Retargeting（動作重定向）轉換

🚨 核心洞察

人形機器人應用的瓶頸不在「捕捉精度」，而在「retargeting 品質」。花 $100K 提升 5mm 捕捉精度，但 retargeting 就吃掉 10mm，完全浪費。投資應放在 retargeting 和 RL 訓練，不是硬體。

八、決策建議

✅ 建議路線

  1. 立即啟動 Phase 0（成本 < NT$5,000）：
    • 安裝 H2O 開源框架（human2humanoid.com）
    • 安裝 GMR（github.com/YanjieZe/GMR）
    • 用 1 台 webcam 驗證 retargeting 品質
    • 確認和椿的機器人型號是否在 GMR 支援列表中（目前 14 種）
  2. Phase 0 成功 → Phase 1：多相機 + OpenPose 提升穩定度
  3. 需要大規模數據 → Phase 2：OptiTrack（非 Vicon）性價比最高
  4. Phase 3 謹慎評估：除非有明確的 ROI，否則不建議 Vicon Markerless

❌ 不建議

  • 直接購買 Vicon Markerless 系統 — 成本 NT$500萬+，ROI 不明確
  • 跳過 retargeting 直接用捕捉數據控制機器人 — 會損壞設備
  • 把 markerless 精度當成機器人控制精度 — 誤差被 retargeting 吃掉
